Дан массив a из n чисел. Требуется написать программу, которая будет отвечать на запросы следующего вида: «для данного k найти минимальный m≥k, что am≥x»

Входные данные
В первой строке входного файла даны три натуральных числа n, m (1⩽n⩽105, 1⩽m⩽107) и a1 (1⩽a1≤n) — количество элементов в массиве, количество запросов и первый элемент массива соответственно. Вторая строка содержит два натуральных числа k1 и x1 (1⩽k1,x1⩽n) — первый запрос.

Элементы a2,a3,…,an задаются следующей формулой:
ai+1=((23⋅ai+21563)modn)+1.
Например, при n=10, a1=5 получается следующий массив: a = (5, 9, 1, 7, 5, 9, 1, 7, 5, 9).

Запросы генерируются следующим образом:

ki+1=((17⋅ki+751+ansi+2i)modn)+1, xi+1=((13⋅xi+593+ansi+5i)modn)+1,
где ansi — ответ на запрос номер i.
Выходные данные
В выходной файл выведите km, xm и ansm (последний запрос и ответ на него).
